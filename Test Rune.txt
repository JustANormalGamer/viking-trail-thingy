Hei.
Dette tekstdokumentet er lastet opp av Rune Holmdal (og sener redigert online).

1. Installer GitHub programvaren.
2. Åpne viking-trail-thingy på GitHub (du er her nå).
3. Logg deg inn med din bruker.
4. Trykk på Clone in Desktop.
5. Lag en ny tekstfil i din lokale GitHub Repository mappe.
   Skriv noe i filen og velg lagre.
6. Når du nå går inn i GitHub programvaren, og trygger på riktig Respository, skal Uncommited changes dukke opp.
   Selekter den/de filene du vil synke med online Respository.
   Velg så Sync i GitHub programvaren.
   NB! NB! Dette fungerer først når Martin har invitert deg som Contributer.
6. Sjekk så på Github at filen din er lastet opp.
7. Yes!
8. Forsøk på opplastingsdemo for klassen.
